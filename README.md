Guess My Number

An interactive number guessing game built with JavaScript where players attempt to guess a randomly generated number within a defined interval. The game provides feedback such as too high or too low, along with score tracking and high score persistence.

Features

    -> Random number generation within a fixed range (1â€“20)
    
    -> Score decreases with incorrect guesses
    
    -> High score tracking
    
    -> Hint system (Too high / Too low / Correct)
    
    -> Replay functionality using the Again button

    -> Dynamic UI updates with DOM manipulation

 Technologies Used

    ->HTML
    
    ->CSS
    
    ->JavaScript (DOM manipulation & event handling)

How to Play

    ->Enter a number between 1 and 20.
    
    ->Click Check to submit your guess.
    
    ->Receive hints to guide your next guess.
    
    ->Score decreases with each wrong attempt.
    
    ->Click Again to restart the game.
